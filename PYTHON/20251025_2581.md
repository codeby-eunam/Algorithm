문제 플랫폼: 백준

문제 이름 또는 번호: 2581 / 소수

링크: https://www.acmicpc.net/submit/2581

오늘의 메모:
```
파이썬 익숙해진줄 알았는데 완전 아녔음.
그래도 친해지는 중~
```

코드:

```
M = int(input())
N = int(input())

primes = []

for num in range(M, N + 1):
    if num < 2:
        continue
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        primes.append(num)

if primes:
    print(sum(primes))
    print(min(primes))
else:
    print(-1)

```
