문제 플랫폼: 백준

문제 이름 또는 번호: 2742 / 기찍 N

링크: [https://www.acmicpc.net/problem/]()9506

오늘의 메모:

안 익숙해서 GPT 도움 받았다...

코드:

```
while True:
    n = int(input())
    if n == -1:  # 입력 종료 조건 (예제 기준)
        break

    divisors = []
    for i in range(1, n):
        if n % i == 0:
            divisors.append(i)

    if sum(divisors) == n:
        print(f"{n} = {' + '.join(map(str, divisors))}")
    else:
        print(f"{n} is NOT perfect.")

```
